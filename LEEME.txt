## Instalación

1. Instalar XAMPP 
2. Abrir phpmyadmin y crear base de datos mysql 'prueba_desis' en tu localhost
3. Importar archivo SQL prueba_desis.sql en la base de datos recien creada
4. En la carpeta C:\xampp\htdocs\ escribir (windows powershell)

```
  git clone https://github.com/pablohost/prueba-desis.git
```
4. En tu navegador abre http://localhost/prueba-desis/

## Tecnologías

**PHP** 8.2.4

**MySQL** 10.4.28-MariaDB / libmysql - mysqlnd 8.2.4

**JQuery** 3.7.1

**XAMPP** 3.3.0
